# MyCarGame - لعبة سباق سيارات 3D احترافية أوفلاين

لعبة سباق سيارات مفتوحة العالم مع بوتات ذكية، متجر سيارات، تخصيص (محرك، ألوان، نيترو)، شرطة تلاحقك، أصوات حقيقية، فيزياء واقعية.

## كيف تشغلها
- افتح في Unity 2022.3+
- بني APK للأندرويد
- استمتع!
